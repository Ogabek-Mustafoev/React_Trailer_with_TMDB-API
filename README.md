<h2>🛠️ Installation Steps:</h2>

<p>1. Download code</p>

```
git clone https://github.com/Ogabek-Mustafoev/React_Trailer_with_TMDB-API.git
```

<p>2. Install packages</p>

```
npm install
```

<p>3. Run server:</p>

```
npm start
```
